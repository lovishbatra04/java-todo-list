# 📝 To-Do List Web App

A simple and clean To-Do List web application built using **HTML, CSS, and JavaScript**.  
This project helps users add, manage, and remove daily tasks in the browser.

---

## 🚀 Features

- Add new tasks
- Delete tasks
- Mark tasks as completed 
- Simple and clean UI

---

## 🛠️ Tech Stack

- HTML5 – Structure
- CSS3 – Styling
- JavaScript – Functionality

---

## 📂 Project Structure
todo-list/
│── index.html
│── style.css
│── script.js


---
## 💡 How It Works

- User enters a task in the input box
- Clicks "Add" button
- Task is added to the list dynamically using JavaScript
- User can delete tasks when done

---

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in any browser
3. Start adding your tasks
